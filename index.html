Programación Web

La programación Web se refiere a la escritura, el marcado y la codificación involucrados en el desarrollo Web, que incluye contenido Web, secuencias de comandos de servidor y cliente Web y seguridad de red.

Los lenguajes más comunes utilizados para la programación Web son XML, HTML, JavaScript, Perl 5 y PHP.
